# Analyze Citation Trends

Analyze citation trends over time for a scholar or group of scholars

## Usage

``` r
AnalyzeCitationTrends(scholar_data, time_period = "all")
```

## Arguments

- scholar_data:

  ScholarData object or list of ScholarProfile objects

- time_period:

  Character string specifying time period ("all", "5y", "10y")

## Value

data.frame with trend analysis
