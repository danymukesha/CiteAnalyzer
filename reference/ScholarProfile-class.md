# ScholarProfile S4 Class

S4 class for storing Google Scholar profile data

## Slots

- `scholar_id`:

  Character string containing the Google Scholar ID

- `name`:

  Character string with scholar's name

- `affiliation`:

  Character string with institutional affiliation

- `interests`:

  Character vector of research interests

- `homepage`:

  Character string with homepage URL

- `citations_total`:

  Numeric total citations

- `citations_5y`:

  Numeric citations in last 5 years

- `h_index`:

  Numeric h-index

- `h_index_5y`:

  Numeric h-index for last 5 years

- `i10_index`:

  Numeric i10-index

- `i10_index_5y`:

  Numeric i10-index for last 5 years

- `publications`:

  data.frame containing publication details
