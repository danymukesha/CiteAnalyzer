# Scholar Profile

Create a comprehensive scholar profile with analysis

## Usage

``` r
ScholarProfile(scholar_id, max_publications = 100, include_network = TRUE)
```

## Arguments

- scholar_id:

  Character string Google Scholar ID

- max_publications:

  Integer maximum number of publications to retrieve

- include_network:

  Logical whether to include collaboration network analysis

## Value

ScholarData object containing complete analysis
